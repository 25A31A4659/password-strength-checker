# Password Strength Checker (C++)

## Description
This project checks the strength of a password using C++.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects digits
- Detects special characters

## How to Run
1. Compile:
   g++ password_checker.cpp -o checker
2. Run:
   ./checker
